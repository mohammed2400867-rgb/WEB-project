📌 Project Description The Restaurant Management System is a dynamic web application designed for seamless dining experiences and real-time management. Customers can order food online, book table reservations for in-house dining, and leave feedback through a dedicated reviews system.

👥 User Roles

Admin: Manages menu items, monitors reservations, and oversees all incoming orders via a dashboard.
Customer: Places food orders, books table reservations, and submits reviews.
Staff: Tracks and updates live order statuses (e.g., Preparing, Ready, Delivered).
⚙ Core Features

Interactive menu with add-to-cart & total calculation
Seamless online order placement & live tracking
In-house table reservation system
Customer review & feedback submission
Role-based access control (Admin, Staff, Customer)
Comprehensive Admin dashboard
